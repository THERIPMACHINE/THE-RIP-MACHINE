# Agent Instructions

Hello, fellow agent! This is a simple Python project. Here are some guidelines to follow when working on this repository:

*   **Project Structure:** The source code is located in the `src` directory, and the tests are in the `tests` directory. Please maintain this separation.
*   **Tests:** Always add tests for new features and ensure that all tests pass before submitting your changes. You can run the tests by executing `python3 -m pytest` from the root of the project.
*   **Documentation:** Keep the `README.md` file up-to-date with any changes to the project.
*   **Code Style:** Follow the PEP 8 style guide for Python code.
*   **Commit Messages:** Write clear and concise commit messages.
*   **Be Perfect:** Strive to make the project even more perfect than you found it.
